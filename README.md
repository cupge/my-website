# CupGe Website

Premium B2B manufacturing website for CupGe, a paper cup and packaging producer.

## Included

- Corporate landing page
- Industrial product catalog
- Quote cart with localStorage
- Request form placeholder
- Production and YouTube section
- Social media links data layer
- Future Google Sheets integration plan

## Run locally

Serve the folder with any static web server, then open the local URL in a browser.

The catalog loads JSON files, so direct double-click opening can be blocked by browser security.

## Structure

- `index.html` - main website shell
- `styles/` - page and component styles
- `js/` - catalog, cart, API, video and social logic
- `data/` - product and social data
- `assets/` - images, icons, fonts
- `docs/` - architecture and Google Sheets planning
